# Básicos de Python

Repositorio estructurado para construir una base sólida en Python mediante ejercicios progresivos, ejemplos aplicados y desarrollo de lógica. Todo el contenido está compuesto por partes creadas personalmente y ejercicios complementarios obtenidos de diversas fuentes para ampliar el entrenamiento práctico.

## Estructura General

### Tema 0 – Instalación del Entorno

Configuración del entorno de trabajo. Instalación de Python, verificación de versiones y primeros pasos ejecutando scripts.

### Tema 1 – Variables, Tipos de Datos y Operaciones

Fundamentos del lenguaje: declaración de variables, tipos primitivos, mutabilidad, operadores y conversión explícita de datos.

### Tema 2 – Condicionales

Construcción de lógica basada en decisiones. Uso de `if`, `elif`, `else` y resolución de caminos múltiples.

### Tema 3 – Listas y Estructuras Iterativas

Manipulación de secuencias. Operaciones principales, recorrido mediante bucles y control de iteraciones.

### Tema 4 – Arrays y Módulos

Introducción al ecosistema modular del lenguaje. Importaciones, uso de módulos estándar y manejo básico de estructuras tipo array.

### Tema 5 – Tuplas y Sets

Colecciones inmutables y colecciones sin duplicados. Operaciones de conjuntos, optimización de búsquedas y relaciones entre elementos.

### Tema 6 – Diccionarios

Estructuras clave–valor. Inserción, actualización, consulta, conteo y extracción de datos para crear miniaplicaciones funcionales.

### Ejercicios

Conjunto de ejercicios para reforzar cada bloque teórico. Incluye programas prácticos como sistemas de votación, registros, cálculos y pequeños proyectos funcionales. Muchos de ellos fueron desarrollados manualmente y otros integran material adaptado desde diversas fuentes educativas.

## Objetivo del Repositorio

Construir dominio real de los fundamentos de Python, desarrollar pensamiento algorítmico y preparar terreno para proyectos intermedios, especializaciones y desarrollo profesional.

## Requisitos

Python 3.10 o superior.
Editor o IDE compatible.

## Ejecución

Clonar el repositorio y ejecutar cualquier archivo `.py` desde terminal o IDE.

```bash
git clone <url-del-repo>
cd Basicos-Python
python archivo.py
```
